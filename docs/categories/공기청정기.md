## 상황별 추천

- **넓은 거실(50㎡ 이상), 소음과 공기질 모두 타협 없이** → **다이슨 빅+콰이엇 BP04**: 대면적 커버에 최대 풍량에서도 56dB 저소음, CO₂ 포함 풀 센서 모니터링이 필요한 최우선 선택
- **30평 내외 넓은 공간, 예산은 절반으로 줄이고 싶다면** → **샤오미 미지아 공기청정기 5**: BP04의 약 40% 수준 가격에 CADR 800㎥/h 대면적 성능 구현, A/S 불가·CA 인증 없음은 감안 필요
- **12~18평 공간, 가성비·낮은 유지비 우선** → **샤오미 미지아 공기청정기 4 프로**: CADR 500㎥/h에 교체 필터 약 3만 원이라는 업계 최저 수준 유지비, 앱 연동까지 갖춘 합리적 선택
- **침실·아이방에서 소음 없이 밤새 가동** → **다이슨 허쉬젯 컴팩트**: 최소 19dB 압도적 저소음과 3.15kg 경량으로 침실 특화, 포름알데히드 제거까지 원스톱 해결
- **원룸·1인 가구·소형 방(4~6평) 입문** → **위닉스 타워 엣지 컴팩트 19.8**: 10만 원대 가격에 CA·KAA 인증과 국내 A/S를 모두 갖춘 소형 공간 최적 선택
- **새집·리모델링 직후 포름알데히드 집중 관리** → **다이슨 허쉬젯 컴팩트**(소형 방) 또는 **다이슨 BP04**(넓은 공간): 두 제품 모두 포름알데히드 전용 필터 탑재, BP04는 촉매 필터로 별도 교체 불필요
- **국내 공식 인증·A/S가 구매 결정의 핵심** → **위닉스 타워 엣지 컴팩트 19.8** 또는 **다이슨(BP04·허쉬젯)**: CA 인증·국내 공식 서비스센터 보유, 샤오미 시리즈는 A/S 제한 주의
- **스마트홈·앱 원격 제어가 필수** → **다이슨(BP04·허쉬젯)** 또는 **샤오미 미지아 시리즈**: MyDyson·Mi Home 앱 원격 제어·공기질 모니터링 지원 (위닉스 타워 엣지 컴팩트는 앱 미지원)
